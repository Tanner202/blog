Minecraft Plugins

March 7, 2024 · 3 min

✨Hello! In this blog post I’m going to be talking about my experience making minecraft plugins. ✨

Story#

I started making plugins in middle school and did a lot during covid. I was always interested in the process of creating a minecraft server and one of the parts involved in making a minecraft server unique and fun was the custom plugins. The reason I wanted to create a minecraft server was because the experiences I had in the past of playing on minecraft servers was so amazing that I had to know what it was like to create those experiences. At first when I was creating a minecraft server I mostly used online plugins and I remember asking my brother to create a plugin for me since he knew how to create minecraft plugins and I didn’t. Eventually I found it more efficient to learn how to create my own plugins rather than begging my brother to work on the plugins. At the time, it was a little overwhelming but I followed an online course on udemy called “Develop Minecraft Plugins (Java)” by Stephen King. The excitement I got when I was able to change the game I loved since I was a child felt amazing. I eventually started working on creating minigames which was the goal I was hoping to achieve to finally bring my ideas to life. Creating minigames involved many different parts and wasn’t as easy as I thought it would be but I created my first minigame which was a classic game of tag. It may sound pretty boring and insignificant but for me at the time this felt like I had done something truely revolutionary! Knowing that this wouldn’t exist without me was part of what made it feel so amazing. I then got to play it with my brother and it was actually quite a bit of fun since I had also built a map for the game and added powerups. After this I then started working on a way more ambitious project based on the server called Island Clash which was a minecraft version of clash of clans with the elements from another popular minecraft gamemode called eggwars. This was my first over ambitious project of many that I didn’t finish. I recently finished another custom plugin which I will share in another article. This was my first ever post so this is all very new to me. Hopefully my experience was somewhat interesting and if you have any comments, questions, or want to collaborate please reach out to me!
